# 神経衰弱ゲーム

## 概要
JavaScriptを使用して制作した1人用ブラウザゲームです。  
時間経過で発生する妨害ミニゲーム（数字あつめゲーム）をクリアしながら、同じ絵柄のカードを見つけていく神経衰弱ゲームになっています。

## 使用技術
 - HTML
 - CSS
 - JavaScript

## 主な機能
### 神経衰弱
 - カードをクリックしてめくる機能
 - 同じ絵柄のカードが揃ったか判定する機能
 - 揃わなかった場合にカードを元に戻す機能
 - 残りのペア数の管理
### 妨害ゲーム（数字あつめゲーム）
 - 数字をクリックするとスコアが加算される機能
 - スコアの管理
 - 一定時間で開始される妨害タイマーの管理

## 工夫した点
 - 配列を使用してカード情報の管理を行いました。
 - 毎回異なるゲーム展開になるよう、カード配置をランダムにしました。
 - 単調にならないよう、一定時間ごとに妨害ミニゲームが発生する仕組みを実装しました。
 - プレイヤーが直感的に操作できるよう、シンプルで分かりやすいUIを意識しました。


## 実行方法
index.htmlをブラウザで開いて実行できます。  
プレイする際は全画面表示を推奨しています。
