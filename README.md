# 神経衰弱ゲーム

## 概要
JavaScriptを使用して制作した1人用ブラウザゲームです。  
時間経過で始まる妨害ミニゲーム（数字あつめゲーム）をクリアしながら、同じ絵柄のカードを見つけていくゲームです。

## 使用技術
 - HTML
 - CSS
 - JavaScript

## 主な機能
### 神経衰弱
 - カードをクリックしてめくる機能
 - 同じ絵柄のカードが揃ったか判定する機能
 - 揃わなかった場合にカードを元に戻す機能
 - 残りのペア数の管理
### 妨害ゲーム（数字あつめゲーム）
 - 数字をクリックするとスコアが加算される機能
 - スコアの管理
 - 妨害タイマーの管理

## 工夫した点
 - 配列を使用し、カード情報の管理を行った
 - 何回でも楽しめるようにカード配置をランダムにした
 - 単調なゲームにならないよう、妨害ゲームを強制的に開始させるタイマーを設定した
 - プレイヤーが直感的に操作できるよう、シンプルなUIを意識した


## 実行方法
index.htmlをブラウザで開いて実行できます。  
全画面表示にしてください。
